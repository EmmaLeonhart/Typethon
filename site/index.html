Typethon

TypeScript syntax. Python runtime ABI. Write algorithmic code in TS, run it on top of numpy, torch, and the rest of the Python ecosystem.

GitHub

What is this?

A transpiler from a restricted subset of TypeScript to Python. The output is not idiomatic Python — it is a compilation artifact that happens to be valid Python so that import numpy just works. Think of TypeScript as the source language and Python as a runtime ABI you target so you can reach the libraries.

Pick an example below, edit the TypeScript on the left, and watch the Python update on the right. Everything runs locally in your browser — no network calls.

File-level pragmas

Declare which Python libraries the file expects with a comment at the top: 

// @typethon target: python
// @typethon import: numpy as np
// @typethon import: pandas as pd

The transpiler emits the matching import statements.

Typethon is early. The output is intentionally crude — it's a compilation target, not maintainable code. Open an issue on GitHub with TypeScript that doesn't transpile cleanly.
